Leafony Leaf

Leaf Template

This project template is the basis of an expansion leaf for the Leafony

and includes both printed circuit board and schematic templates.

PCB Template

The PCB template features the same dimensions as
other Leafony Leaves and includes a unified connector
to interface with other Leaves.

Schematic Template

The schematic template includes the connectors corresponding symbol
and purpose based pin naming.
