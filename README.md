# cad-template

This repository is a template library of Leafony PCB design.

![brd.png](img/brd.png)

## Supported CAD tools
* KiCad 5.0.0 or later

## Usage
### KiCad

1. Open KiCad.
1. `File -> New -> Project` from Template.
1. `Browse` this repository and select Leafony template.
